# Mogadishu Rentals (mobile-first demo)

A lightweight, mobile-first real estate rental UI built with React + Vite. Browse, save, and list rentals (apartments, houses, rooms, commercial, hotels) around Mogadishu.

## Run Locally

Prerequisites: Node.js

1) Install dependencies
```
npm install
```
2) Run the app
```
npm run dev
```
3) Build for production
```
npm run build
```

Environment: optional `.env.local` for any API keys you add later.
